Atualizar Versão

 

Caso seja necessário atualizar a versão do Endereçador, o usuário deverá selecionar o menu "Ajuda" e em seguida "Atualizar versão", sendo que se o usuário tiver conectado com a Internet automaticamente o sistema apresentará o site para baixar a versão atualizada do sistema, como apresentado nas telas abaixo:  

 

 

 

Voltar para o Menu Ajuda
