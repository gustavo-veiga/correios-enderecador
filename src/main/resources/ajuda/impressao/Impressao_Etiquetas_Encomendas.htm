Impressão Etiquetas Encomendas

Para elaboração de etiquetas para encomendas clique no Menu Etiquetas, opção Encomendas ou então clique no botão da barra de ferramentas Etiquetas para Encomendas. A tela abaixo será aberta:

 

 

 

 

Será apresentada a tela para preenchimento dos campos necessários para impressão das etiquetas de encomendas, onde os campos devem ser preenchidos de acordo com a necessidade do usuário;

Escolha o tamanho do rótulo que você deseja marcando uma das opções: 4 ou 2 rótulos por folha. Na opção de Incluir telefone nas etiquetas, o número do telefone irá aparecer nas etiquetas caso essa opção seja marcada, tanto para o remetente, quando para o destinatário. Selecione o remetente, informe a partir de qual etiqueta deseja imprimir e depois selecione os Destinatários. O programa permite a seleção de um ou mais destinatários através do Botão Selecionar Destinatário e também permite selecionar um ou mais Grupos de Destinatários através do botão Selecionar Grupos. Mesmo que o Destinatário já esteja na lista de Destinatários ele não será repetido. Se desejar remover um determinado Destinatário clique no botão Remover Destinatário. Se deseja limpar toda a relação de Destinatários, clique no botão Remover Todos.

 

 

O usuário deverá iniciar o processo selecionando o destinatário para o qual deseja enviar as encomendas, ou selecionar o grupo, no caso de enviar a vários destinatário, como serão mostrados nas telas a seguir:

 

 

 

Na opção de "Selecionar Destinatário", o usuário deverá selecionar o destinatário que deseja e "Confirmar";

Na opção de Selecionar Grupo", o procedimento será o mesmo;

Após serem confirmados os destinatários, será apresentado na tela a lista com os destinatários selecionados pelo usuário. Se houver necessidade de exclusão de algum destinatário o usuário deverá selecionar a opção "Excluir Destinatário" ou "Excluir todos", conforme necessidade, bastando apenas selecionar o(s) destinatário(s) desejados e clicar sobre uma das opções citadas;

 

 

Para visualizar o modelo de "Aviso de Recebimento", selecionar a opção "Visualizar AR":

 

 

 

Será apresentado a seguinte janela:

 

 

 

Para visualizar o modelo de impressão das etiquetas, selecionar a opção "Visualizar Etiquetas":

 

 

Será apresentado a seguinte janela:

 

 

 

Observação: As etiquetas poderão ser impressas em papel comum (A4, carta), não necessariamente em folhas próprias para impressão de etiquetas.
