Impressão Envelopes

Para imprimir diretamente em envelopes o usuário deverá seguir os seguintes passos:

Selecionar a opção "Impressão >> Envelopes", como mostrado na janela a seguir:

 

 

Ou se preferir o usuário poderá usar o ícone apresentado na tela principal do sistema;

 

 

Será apresentada a tela para preenchimento dos campos necessários para impressão dos envelopes, onde os campos devem ser preenchidos de acordo com a necessidade do usuário;

 

 

 

O usuário deverá iniciar o processo selecionando o destinatário ou selecionar o grupo, no caso de enviar a vários destinatários, como serão mostrados nas telas a seguir:

 

 

 

 

Na opção de "Selecionar Destinatário", o usuário deverá selecionar o destinatário que deseja e "Confirmar";

Na opção de Selecionar Grupo", o procedimento será o mesmo;

Após serem confirmados os destinatários, será apresentado na tela a lista com os destinatários selecionados pelo usuário. Se houver necessidade de exclusão de algum destinatário o usuário deverá selecionar a opção "Excluir Destinatário" ou "Excluir todos", conforme necessidade, bastando apenas selecionar o(s) destinatário(s) desejados e clicar sobre uma das opções citadas;
