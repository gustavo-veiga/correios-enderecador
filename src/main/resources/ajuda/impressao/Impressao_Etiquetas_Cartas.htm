Impressão Etiquetas Cartas

Para imprimir etiquetas para cartas o usuário deverá seguir os seguintes passos:

Selecionar a opção "Impressão >> Etiquetas >> Cartas", como mostrado na janela a seguir:

 

 

Ou se preferir o usuário poderá usar o ícone apresentado na tela principal do sistema;

 

 

Será apresentada a tela para preenchimento dos campos necessários para impressão das etiquetas de cartas, onde os campos devem ser preenchidos de acordo com a necessidade do usuário;

 

 

Ao selecionar a opção "Remetente", a tela oferece apenas os campos para escolher remetentes, selecionando "Destinatário", oferecerá apenas opções para cadastrar destinatário, se o usuário desejar inserir remetente e destinatário deve optar pela opção "Remetente e Destinatário", como mostrado na janela abaixo:

 

 

O usuário deverá iniciar o processo selecionando o destinatário para o qual deseja enviar as cartas, ou selecionar o grupo, no caso de enviar a vários destinatário, como serão mostrados nas telas a seguir:

 

 

 

 

Na opção de "Selecionar Destinatário", o usuário deverá selecionar o destinatário que deseja e "Confirmar";

Na opção de Selecionar Grupo", o procedimento será o mesmo;

Após serem confirmados os destinatários, será apresentado na tela a lista com os destinatários selecionados pelo usuário. Se houver necessidade de exclusão de algum destinatário o usuário deverá selecionar a opção "Excluir Destinatário" ou "Excluir todos", conforme necessidade, bastando apenas selecionar o(s) destinatário(s) desejados e clicar sobre uma das opções citadas;

 

 

Para visualizar o modelo de "Aviso de Recebimento", selecionar a opção "Visualizar AR":

 

 

 

Será apresentado a seguinte janela:

 

 

 

Para visualizar o modelo de impressão das etiquetas, selecionar a opção "Visualizar Etiquetas":

 

 

Será apresentado a seguinte janela:

 

 

Observação: As etiquetas poderão ser impressas em papel comum (A4, carta), não necessariamente em folhas próprias para impressão de etiquetas.
