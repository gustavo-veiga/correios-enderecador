Impressão

O menu impressão oferece ao usuário a opção de impressão de etiquetas e envelopes padronizados de acordo com as normas de endereçamento dos Correios conforme as opções:

 

Etiquetas para Encomendas permite a impressão de etiquetas para encomendas, oferecendo opções de modelos, tamanhos, visualização das etiquetas e dados de remetente e destinatário;

Etiquetas para Cartas, permite a impressão de etiquetas para cartas, oferecendo opções de modelos, tamanhos, visualização das etiquetas e dados de remetente e destinatário;

Impressão Direta em Envelopes, permite a impressão direta em envelopes, oferecendo ao usuário opções de modelos, tamanhos e dados de remetente e destinatário;

Etiquetas - tipos de etiquetas utilizadas pelo Endereçador;

Envelopes - formatos admitidos;
