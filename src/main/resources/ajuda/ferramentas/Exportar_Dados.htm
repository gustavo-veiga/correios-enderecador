Exportar Dados

 

Para exportar dados de Destinatários para arquivos do tipo .TXT e .CSV clique no Menu Ferramentas, opção Exportar Dados ou então clique no botão Exportar Dados da barra de ferramentas. Quando a tela de exportação de dados for aberta siga os passos:  

 

 

 

Será apresentada então a seguinte tela:

 

 

  1. Selecione os destinatários que deseja exportar.

  2. Clique no botão exportar.

  3. Informe o nome e a extensão do arquivo que será gerado (a extensão do arquivo deve ser TXT ou CSV).

  4. Clique em Salvar.

 

 

 

Voltará então para a tela principal de Exportar Dados, onde o usuário informará o endereço para o qual deseja exportar os dados selecionados anteriormente:

 

 

O usuário deve selecionar o diretório para o qual deseja exportar os dados, deverá informar um nome para o novo arquivo e clicar sobre a opção "Save", como mostrada na tela acima, e receberá a seguinte mensagem como confirmação:

 

 

A estrutura de cada registro do arquivo gerado será composta de 15 campos todos obrigatoriamente separados por ponto e vírgula ";" seguindo a seguinte ordem:

  1. Tratamento: campo opcional, pode ficar em branco.

  2. Empresa/Nome (linha 1): campo obrigatório.

  3. Empresa/Nome (linha 2): campo opcional, pode ficar em branco;

  4. Caixa Postal: campo opcional, pode ficar em branco.

  5. Endereço: campo obrigatório.

  6. Número/Lote: campo obrigatório.

  7. Complemento: opcional, pode ficar em branco.

  8. Bairro: campo opcional, pode ficar em branco.

  9. Cidade: campo obrigatório.

  10. UF: campo obrigatório.

  11. E-mail: campo opcional, pode ficar em branco, porém se for preenchido será validado.

  12. Telefone: campo opcional, pode ficar em branco.

  13. Fax: campo opcional, pode ficar em branco.

  14. CEP Caixa Postal: campo opcional, pode ficar em branco.

  15. CEP: (campo obrigatório).

Voltar para o Menu Ferramentas
