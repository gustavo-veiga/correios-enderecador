Configurar Usuário de Rede

Se a opção Usar um servidor proxy tiver sido ativada no menu Configurar Proxy, então o login de rede e a senha deverão ser informados para autenticação no servidor Proxy.

 

Selecionar o menu Ferramentas, e em seguida a opção "Configurar Usuário de Rede":

 

 

Será apresentada a seguinte tela:

 

 

O login do usuário e senha poderão ser informados diretamente dessa forma, ou então na hora de buscar o CEP pela primeira vez no sistema. Toda vez que o programa for acessado, essa configuração deverá ser preenchida, pois o programa não armazena nenhuma informação do usuário.

 

 

Voltar para o Menu Ferramentas
