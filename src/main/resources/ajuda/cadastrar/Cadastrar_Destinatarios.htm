Cadastrar Destinatário

Para cadastrar um destinatário, o usuário deverá preencher os campos da tela Cadastro de Destinatário, conforme figura abaixo:

 

Selecionar a opção Cadastro, e em seguida "Cadastrar  Destinatários", como mostrada na tela abaixo:

 

 

Se preferir, o usuário poderá selecionar o botão "Cadastrar Destinatário" encontrado na tela principal;

 

 

Após clicar sobre "Cadastrar Destinatário", será apresentada a seguinte tela:

 

 

Para efetuar um novo cadastro, selecionar a opção "Novo Destinatário", será apresentada a tela para preenchimento dos campos;

Os campos preenchidos de amarelo tem preenchimento obrigatório;

 

 

 

O usuário pode usar a opção de "Capturar Endereço", basta colocar o CEP correto e clicar na opção, no caso de não saber o CEP, poderá ir à opção "Não sei o CEP", o qual oferece  a opção de pesquisa; Observação: A opção de "Captura de Endereço", só estará disponível se a "Chave de Registro" for cadastrada;

Para a confirmação do cadastro do novo destinatário, o usuário deverá selecionar a opção de "Confirmar" para que os dados sejam adicionados a lista de destinatários e receberá a seguinte confirmação:

 

 

A tela disponibiliza também as opções de "Limpar Tela", a qual limpa os campos disponibilizando novos campos para preenchimento, e "Voltar", a qual permite que o usuário retorne a tela de cadastrar destinatário;

Após efetuar o cadastro, a tela "Cadastro de Destinatários" oferece as opções "Editar", "Pesquisar" e "Excluir", como apresentado na tela a seguir:

 

 

A opção "Editar" permite ao usuário que seja realizado qualquer alteração em um Destinatário já cadastrado anteriormente;

 

 

Após ser selecionado a opção "Editar", será apresentada a seguinte tela, para que as alterações possam ser efetuadas;

 

 

Se ao selecionar a opção "Editar" não houver nenhum destinatário selecionado, será apresentada a seguinte mensagem:

 

 

Após fazer as alterações que forem necessárias, o usuário deverá confirmar a alteração para que seja gravada, e receberá a seguinte mensagem:

 

 

    A opção "Pesquisar" permite ao usuário que realize pesquisa por nome do destinatário, basta digitar o nome do destinatário que deseja pesquisar e clicar no botão, como mostrado na tela abaixo:

 

 

A opção "Excluir" permite que seja excluído algum destinatário, caso seja necessário, bastando apenas que seja selecionado o nome do destinatário, na lista mostrada abaixo, e em seguida escolher a opção "Excluir";

 

 

Será apresentada uma tela para confirmação de exclusão;

 

 

Se o usuário realmente desejar excluir os dados, deverá escolher a opção "Sim", e será efetuada a exclusão;
