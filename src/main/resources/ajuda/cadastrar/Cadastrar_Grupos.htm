Cadastrar Grupos

Para efetuar o Cadastro de Grupos, o usuário deverá seguir os seguintes passos:

 

Selecione a opção Cadastro e em seguida "Cadastrar Grupos";

 

 

Se preferir o usuário poderá selecionar a opção através do ícone que se encontra na tela principal;

 

 

Será apresentado então a tela de opções para cadastro de novos grupos;

O usuário deverá selecionar a opção "Novo Grupo';

 

 

No campo "Grupo", deverá ser informado o nome que o usuário deseja cadastrar o novo grupo;

 

 

Após informar o nome do grupo, o usuário deverá selecionar na lista apresentada, quais destinatários deseja inserir no grupo que está sendo criado, como mostrado no exemplo abaixo;

 

 

Após selecionar os destinatários na lista apresentada, o usuário deverá clicar na opção "Adicionar", inserindo assim os dados no grupo cadastrado;

 

 

Quando os dados forem adicionados, deverá ser selecionada a opção "Confirmar" para finalizar a operação, e receberá a seguinte mensagem:

 

 

Se for necessário alguma alteração em um grupo já cadastrado, o usuário deve informar o nome do grupo que deseja alterar, e após clicar na opção "Editar", a qual oferece as funções de inserir, alterar e remover dados do grupo selecionado;

 

 

No caso do usuário possuir uma lista extensa de grupos cadastrados, poderá utilizar a opção "Pesquisar", informando o nome do grupo que deseja procurar e clicando sobre a opção !Pesquisar";

 

 

A opção "Excluir" permite que seja excluído algum grupo cadastrado, caso seja necessário, bastando apenas que seja selecionado o nome do grupo, na lista mostrada abaixo, ou informado no campo "Procurar por" o nome do grupo desejado e em seguida escolher a opção "Excluir";
