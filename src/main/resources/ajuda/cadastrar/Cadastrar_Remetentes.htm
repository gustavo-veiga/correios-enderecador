Cadastrar Remetentes

 

Para efetuar o Cadastro de Remetentes, o usuário deverá seguir os seguintes passos:

 

Selecione a opção Cadastro e em seguida "Cadastrar Remetentes";

 

 

Se preferir, o usuário poderá selecionar a opção pressionando o botão de atalho que se encontra na tela principal;

 

 

 

Será apresentado então a tela de opções para cadastro de remetentes;

O usuário deverá selecionar a opção "Novo Remetente";

 

 

Informe os dados para o preenchimento dos campos abaixo, (os campos amarelos são obrigatórios);

 

 

O usuário pode usar a opção de "Capturar Endereço", basta colocar o CEP correto e clicar na opção, no caso de não saber o CEP, poderá ir à opção "Não sei o CEP", o qual oferece  a opção de pesquisa; Observação: A opção de "Captura de Endereço", só estará disponível se a "Chave de Registro" for cadastrada;

Para a confirmação do cadastro do novo remetente, o usuário deverá selecionar a opção de "Confirmar" para que os dados sejam adicionados a lista de remetentes e receberá a seguinte confirmação:

 

 

A tela disponibiliza também as opções de "Limpar Tela", a qual limpa os campos disponibilizando novos campos para preenchimento, e "Voltar", a qual permite que o usuário retorne a tela de cadastrar usuário;

Após efetuar o cadastro, a tela "Cadastro de Remetentes" oferece as opções "Editar", "Pesquisar" e "Excluir", como apresentado na tela a seguir:

 

 

A opção "Editar" permite ao usuário que seja realizado qualquer alteração em um Remetente já cadastrado anteriormente;

 

 

Após ser selecionado a opção "Editar", será apresentada a seguinte tela, para que as alterações possam ser efetuadas;

 

 

Se ao selecionar a opção "Editar" não houver nenhum remetente selecionado, será apresentada a seguinte mensagem:

 

 

Após fazer as alterações que forem necessárias, o usuário deverá confirmar a alteração para que seja gravada, e receberá a seguinte mensagem:

 

 

A opção "Pesquisar" permite ao usuário que realize pesquisa por nome do remetente, basta digitar o nome do remetente que deseja pesquisar e clicar no botão, como mostrado na tela abaixo:

 

 

A opção "Excluir" permite que seja excluído algum remetente, caso seja necessário, bastando apenas que seja selecionado o nome do remetente, na lista mostrada abaixo, e em seguida escolher a opção "Excluir";

 

 

Será apresentada uma tela para confirmação de exclusão;

 

 

Se o usuário realmente desejar excluir os dados, deverá escolher a opção "Sim", e será efetuada a exclusão;
