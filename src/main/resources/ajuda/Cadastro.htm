Cadastro

No módulo Cadastro estão disponíveis os cadastros de remetentes, destinatários e grupos com os dados necessários para preencher as etiquetas de endereçamento disponíveis no Endereçador.

 

Cadastrar Remetentes, opção que permite incluir, alterar ou excluir dados referentes ao(s) remetente(s);

Cadastrar Destinatários,  opção que permite incluir, alterar ou excluir dados referentes a destinatários;

Cadastrar Grupos, opção que permite incluir, alterar ou excluir grupos de destinatários. Os grupos podem reunir uma lista de destinatários para facilitar a seleção de destinatários;
